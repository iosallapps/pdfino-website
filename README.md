## üöÄ Quick Start

### Deploy to GitHub Pages (FREE) - 5 Minutes Setup

#### Step 1: Create GitHub Repository
1. Go to [github.com](https://github.com) and log in
2. Click the "+" icon ‚Üí "New repository"
3. Name it `pdfino-website` (or any name you like)
4. Make it **Public** (required for free GitHub Pages)
5. Click "Create repository"

#### Step 2: Upload All Files
1. In your new repository, click "uploading an existing file"
2. Drag and drop ALL files from this `docs` folder:
   - `index.html`
   - `terms.html`
   - `privacy.html`
   - `app-ads.txt` ‚ö†Ô∏è **IMPORTANT for AdMob!**
3. Click "Commit changes"

#### Step 3: Enable GitHub Pages
1. Go to repository "Settings" ‚Üí "Pages" (in left sidebar)
2. Under "Source", select:
   - Branch: `main`
   - Folder: `/ (root)`
3. Click "Save"
4. Wait 1-2 minutes for deployment

#### Step 4: Get Your Website URL
Your site will be live at:
```
https://YOUR-USERNAME.github.io/pdfino-website/
```

Replace `YOUR-USERNAME` with your GitHub username.

## üî¥ CRITICAL: Update AdMob

After deployment, you MUST update AdMob with your website URL:

1. Go to [AdMob Console](https://apps.admob.com)
2. Select "Pdfino iOS"
3. Go to "App settings"
4. Add your GitHub Pages URL: `https://YOUR-USERNAME.github.io/pdfino-website/`
5. Save and wait for verification (can take a few hours)

## üî¥ CRITICAL: Update App Store Connect

You need to update the URLs in your app:

1. Go to App Store Connect
2. Under "App Information" ‚Üí add your website URL
3. The AdMob verification will check this URL for the `app-ads.txt` file
